What is the difference between var, let, and const?
var: var is a method to declare variable. It hoisted and set value undefind. After es-6 it is obsolet.

let: let is a keyword to declare variable in es-6. when we need to reassign variable value we have to use let. let is hoisted but it does not set value. it stay in temporal dead zone.

const: const is a keyword for declarering fix values. if used const, the value can not be reassign or any change.

What is the spread operator (...)?
spread operator is method for copying an array and object. it copies array element and object properties without reference.

What is the difference between map(), filter(), and forEach()?
map() = map is an array method which loop in every element of array with a condition. Then returns conditional result.

filter() = filter loop in an array. It check every element with condition and return all matched element.

forEach() = forEach method also loop in an array with condition. But it does not return anything.
